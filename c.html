Introduction

C is a powerful, general-purpose programming language that has been the foundation of modern computing for decades. Developed in the early 1970s by Dennis Ritchie at Bell Labs, C provides low-level memory access, high efficiency, and portability, making it ideal for system programming.

One of C's most significant contributions is its role in developing operating systems, including UNIX and modern OS kernels like Linux. Its simplicity, speed, and flexibility have made it a preferred choice for embedded systems, game development, and high-performance applications.

Despite being one of the oldest programming languages, C remains widely used today. It has influenced many other languages, including C++, Java, and Python. Mastering C is a fundamental step for any programmer looking to understand how computers work at a low level and build efficient software.

Operators

Operators in C allow you to perform various computations and manipulations on data. This section covers:

  • Arithmetic Operators – Perform basic mathematical operations like addition, subtraction, multiplication, and division.
  • Logical Operators – Used for decision-making (AND, OR, NOT).
  • Bitwise Operators – Manipulate individual bits in variables.
  • Relational Operators – Compare values (greater than, less than, equal to, etc.).
  • Assignment and Miscellaneous Operators – Modify variables and perform type casting.
                #include 

                    int main() {
                        // Arithmetic Operators
                        int a = 10, b = 3;
                        printf("Addition: %d\n", a + b);
                        printf("Subtraction: %d\n", a - b);
                        printf("Multiplication: %d\n", a * b);
                        printf("Division: %d\n", a / b);
                        printf("Remainder: %d\n", a % b);
                    
                        // Relational Operators
                        printf("Is a > b? %d\n", a > b);
                        printf("Is a == b? %d\n", a == b);
                        printf("Is a != b? %d\n", a != b);
                    
                        // Logical Operators
                        int x = 1, y = 0;
                        printf("Logical AND: %d\n", x && y);
                        printf("Logical OR: %d\n", x || y);
                        printf("Logical NOT: %d\n", !x);
                    
                        // Assignment Operators
                        int c = 5;
                        c += 3; // c = c + 3
                        printf("Assignment (c += 3): %d\n", c);
                    
                        // Increment and Decrement Operators
                        int d = 7;
                        printf("Post-increment: %d\n", d++);
                        printf("Pre-increment: %d\n", ++d);
                    
                        return 0;
                    }

Selection (Decision Making)

Decision-making constructs help programs execute specific blocks of code based on conditions. This section explores:

  • if-else statements – Execute different code blocks based on conditions.
  • switch-case – An alternative to multiple if-else statements for handling multiple conditions.
  • Conditional (Ternary) Operator – A shorthand way to write simple if-else statements.
                #include 

                    int main() {
                        int num;
                    
                        // Input from user
                        printf("Enter a number: ");
                        scanf("%d", &num);
                    
                        // if statement
                        if (num > 0) {
                            printf("%d is positive.\n", num);
                        }
                    
                        // if-else statement
                        if (num % 2 == 0) {
                            printf("%d is even.\n", num);
                        } else {
                            printf("%d is odd.\n", num);
                        }
                    
                        // else-if ladder
                        if (num > 0) {
                            printf("%d is positive.\n", num);
                        } else if (num < 0) {
                            printf("%d is negative.\n", num);
                        } else {
                            printf("%d is zero.\n", num);
                        }
                    
                        // switch-case statement
                        switch (num) {
                            case 1:
                                printf("You entered one.\n");
                                break;
                            case 2:
                                printf("You entered two.\n");
                                break;
                            case 3:
                                printf("You entered three.\n");
                                break;
                            default:
                                printf("You entered a number other than 1, 2, or 3.\n");
                                break;
                        }
                    
                        return 0;
                    }

Iteration (Loops)

Loops in C allow repeated execution of a block of code. This section covers:

  • for loop – Used when the number of iterations is known.
  • while loop – Executes as long as a condition remains true.
  • do-while loop – Similar to a while loop but guarantees at least one execution.
                #include 

                    int main() {
                        int i;
                    
                        // for loop
                        printf("For Loop:\n");
                        for (i = 1; i <= 5; i++) {
                            printf("%d ", i);
                        }
                        printf("\n\n");
                    
                        // while loop
                        printf("While Loop:\n");
                        i = 1;
                        while (i <= 5) {
                            printf("%d ", i);
                            i++;
                        }
                        printf("\n\n");
                    
                        // do-while loop
                        printf("Do-While Loop:\n");
                        i = 1;
                        do {
                            printf("%d ", i);
                            i++;
                        } while (i <= 5);
                        printf("\n\n");
                    
                        // Nested loops
                        printf("Nested Loops (Pattern):\n");
                        for (i = 1; i <= 5; i++) {
                            for (int j = 1; j <= i; j++) {
                                printf("* ");
                            }
                            printf("\n");
                        }
                    
                        return 0;
                    }

Storage Classes

Storage classes define the scope, visibility, and lifetime of variables in C. This section discusses:

  • auto – Default storage class for local variables.
  • static – Retains variable value even after function execution ends.
  • extern – Declares variables globally across multiple files.
  • register – Suggests storage in CPU registers for faster access.
                #include 

                    // Global variable with static storage class
                    static int globalStaticVar = 10;
                    
                    // Function to demonstrate automatic storage class
                    void autoStorageClass() {
                        // Automatic (local) variable
                        auto int autoVar = 20;
                        printf("Auto Variable: %d\n", autoVar);
                    }
                    
                    // Function to demonstrate register storage class
                    void registerStorageClass() {
                        // Register variable
                        register int registerVar = 30;
                        printf("Register Variable: %d\n", registerVar);
                    }
                    
                    // Function to demonstrate static storage class
                    void staticStorageClass() {
                        // Static variable (retains its value between function calls)
                        static int staticVar = 40;
                        printf("Static Variable: %d\n", staticVar);
                        staticVar++;
                    }
                    
                    // Function to demonstrate extern storage class
                    void externStorageClass() {
                        // Extern variable (refers to the global variable)
                        extern int globalStaticVar;
                        printf("Extern Variable: %d\n", globalStaticVar);
                    }
                    
                    int main() {
                        printf("===== Storage Classes in C =====\n");
                    
                        // Automatic Storage Class
                        printf("\nAuto Storage Class:\n");
                        autoStorageClass();
                    
                        // Register Storage Class
                        printf("\nRegister Storage Class:\n");
                        registerStorageClass();
                    
                        // Static Storage Class
                        printf("\nStatic Storage Class:\n");
                        staticStorageClass();
                        staticStorageClass(); // Value retained between calls
                    
                        // Extern Storage Class
                        printf("\nExtern Storage Class:\n");
                        externStorageClass();
                    
                        return 0;
                    }

Arrays

Arrays store multiple values of the same type under a single variable name. Learn about:

  • One-dimensional arrays – Store linear lists of values.
  • Multi-dimensional arrays – Store tabular or matrix-like data.
  • Array manipulation – Learn sorting, searching, and other operations.
                #include 

                    int main() {
                        // Array declaration and initialization
                        int numbers[5] = {10, 20, 30, 40, 50};
                    
                        // Accessing array elements
                        printf("Array elements:\n");
                        for (int i = 0; i < 5; i++) {
                            printf("numbers[%d] = %d\n", i, numbers[i]);
                        }
                    
                        // Modifying array elements
                        numbers[2] = 100; // Change the third element
                        printf("\nAfter modification:\n");
                        printf("numbers[2] = %d\n", numbers[2]);
                    
                        // Sum of array elements
                        int sum = 0;
                        for (int i = 0; i < 5; i++) {
                            sum += numbers[i];
                        }
                        printf("\nSum of array elements: %d\n", sum);
                    
                        // 2D array example
                        int matrix[2][3] = {
                            {1, 2, 3},
                            {4, 5, 6}
                        };
                        printf("\n2D Array (Matrix):\n");
                        for (int i = 0; i < 2; i++) {
                            for (int j = 0; j < 3; j++) {
                                printf("%d ", matrix[i][j]);
                            }
                            printf("\n");
                        }
                    
                        return 0;
                    }

Strings

Strings in C are represented as character arrays. This section covers:

  • Character arrays – Store sequences of characters.
  • String manipulation – Learn common functions like strcpy, strcat, strlen, and strcmp.
                #include 
                    #include

Structures

Structures allow grouping of different data types into a single unit. This section explores:

  • Structures – Define custom data types using struct.
  • Unions – Store different types of data in the same memory location.
  • typedef – Create type aliases for better code readability.
                #include 
                    #include

                    struct Student {
                        char name[50];
                        int age;
                        float gpa;
                    };
                    
                    int main() {
                        // Declare a structure variable
                        struct Student student1;
                    
                        // Assign values to structure members
                        strcpy(student1.name, "Alice");

                        student1.gpa = 3.8;
                    
                        // Access and print structure members
                        printf("Student 1:\n");
                        printf("Name: %s\n", student1.name);

                        printf("GPA: %.2f\n", student1.gpa);
                    
                        // Declare and initialize a structure variable
                        struct Student student2 = {"Bob", 22, 3.5};
                    
                        // Access and print structure members
                        printf("\nStudent 2:\n");
                        printf("Name: %s\n", student2.name);
                        printf("Age: %d\n", student2.age);
                        printf("GPA: %.2f\n", student2.gpa);
                    
                        // Array of structures

                        // Access and print array of structures

                            printf("Name: %s\n", students[i].name);
                            printf("Age: %d\n", students[i].age);
                            printf("GPA: %.2f\n", students[i].gpa);
                        }
                    
                        return 0;
                    }

File Manipulation

File handling enables reading and writing data to files. Learn about:

  • File operations – Open, close, read, and write files.
  • File handling functions – Use fopen, fclose, fread, fwrite, fprintf, and fscanf.
  • Binary vs Text Files – Understand different file storage formats.
                #include 

                    int main() {
                        FILE *file;
                        char data[100];
                    
                        // Writing to a file
                        file = fopen("example.txt", "w");
                        if (file == NULL) {
                            printf("Error opening file!\n");
                            return 1;
                        }
                        printf("Enter text to write to the file: ");
                        fgets(data, sizeof(data), stdin);
                        fprintf(file, "%s", data);
                        fclose(file);
                        printf("Data written to file successfully.\n\n");
                    
                        // Reading from a file
                        file = fopen("example.txt", "r");
                        if (file == NULL) {
                            printf("Error opening file!\n");
                            return 1;
                        }
                        printf("Reading from file:\n");
                        while (fgets(data, sizeof(data), file) != NULL) {
                            printf("%s", data);
                        }
                        fclose(file);
                    
                        // Appending to a file
                        file = fopen("example.txt", "a");
                        if (file == NULL) {
                            printf("Error opening file!\n");
                            return 1;
                        }
                        printf("\nEnter text to append to the file: ");
                        fgets(data, sizeof(data), stdin);
                        fprintf(file, "%s", data);
                        fclose(file);
                        printf("Data appended to file successfully.\n\n");
                    
                        // Reading the updated file
                        file = fopen("example.txt", "r");
                        if (file == NULL) {
                            printf("Error opening file!\n");
                            return 1;
                        }
                        printf("Updated file content:\n");
                        while (fgets(data, sizeof(data), file) != NULL) {
                            printf("%s", data);
                        }
                        fclose(file);
                    
                        return 0;
                    }

Error Handling

Debugging and error handling are crucial for writing robust programs. This section covers:

  • Common runtime errors – Understand segmentation faults, memory leaks, and undefined behaviors.
  • Debugging techniques – Learn to use gdb and debugging tools to trace errors.
  • Error-handling functions – Use errno and perror() to handle errors.
                #include 
                    #include   // For errno
                    #include  // For strerror
                    
                    int divide(int a, int b, int *result) {
                        if (b == 0) {
                            return -1; // Error: Division by zero
                        }
                        *result = a / b;
                        return 0; // Success
                    }
                    
                    int main() {
                        // Example 1: Handling division by zero
                        int result;
                        if (divide(10, 0, &result) == -1) {
                            printf("Error: Division by zero is not allowed.\n");
                        } else {
                            printf("Result of division: %d\n", result);
                        }
                    
                        // Example 2: Handling file errors
                        FILE *file = fopen("non_existent_file.txt", "r");
                        if (file == NULL) {
                            printf("Error opening file: %s\n", strerror(errno)); // Print error message
                        } else {
                            fclose(file);
                        }
                    
                        // Example 3: Using perror for error handling
                        file = fopen("another_non_existent_file.txt", "r");
                        if (file == NULL) {
                            perror("Error opening file"); // Print error message with custom prefix
                        } else {
                            fclose(file);
                        }
                    
                        return 0;
                    }
            
                Error: Division by zero is not allowed.
